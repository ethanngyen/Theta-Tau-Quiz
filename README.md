# Theta Tau Study Guide

A simple site for studying.  
Users unlock the page with a single password (no accounts), then answer questions by typing into a textbox. Answers are marked **green** if correct and **red** if incorrect.

## Features
- 🔒 **Single password gate** (no accounts)
- ⌨️ **Type-to-answer** inputs (press **Enter** or click **Check**)
- ✅ Correct = green, ❌ Wrong = red
- 🔎 Search questions/answers
- 🔁 **Reload quiz** button (refreshes the page)
- ➕ Easy to add more questions in one place (the `questions` array)
